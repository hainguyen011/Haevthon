## Abstract
Research on: Hackathon Handbook PDF content and design best practices for a premium tech event like Haevthon 2026.

## Insights
- **Web Search (hackathon.guide, openhackathons.org)**: Hackathon Handbook Best Practices:
1. Essential Content: Event Overview, Timeline, Rules/CoC, Deliverables (submission requirements), Logistics/Resources, Mentorship info, and Judging Criteria.
2. Design: Single source of truth, clickable TOC, visual hierarchy for quick skimming, and templates/checklists to reduce friction.
3. Digital-First: While PDF is desired, a web-based version that is printable ensures real-time updates and accessibility.
4. Haevthon Context: Integrate the premium 'Aevum' aesthetic (glassmorphism) and clear technical tracks (AI, Infrastructure, etc.).
