## Quy ước Thư viện (Research Storage Conventions)

1. **Naming**: Đặt tên file có tiền tố (`arch_`, `tech_`, `evol_`, `sec_`).
2. **YAML Frontmatter**: Phải có metadata (title, date, author_persona, tags, status) ở đầu mỗi file.
3. **Indexing**: Mọi báo cáo mới phải được chèn link vào danh sách bên dưới.
